# Backlink KPI Dashboard

What success means, and the running log. **No vanity metrics:** DA of acquired
links is recorded but never the goal — leads, sales, and rankings are.

| KPI | Source | Target (first 90 days) |
|---|---|---|
| Quality links acquired (policy-compliant, live, verified) | CSV `won` rows | SGD 8–12 · Z&E 6–10 · Trojan 4–6 (entity + reviews) |

| Outreach conversion rate | CSV: won ÷ contacted | ≥10% overall (partnerships/vendor lanes run higher; editorial lower) |

| Assisted conversions / leads influenced | GA4 + CRM lead sources; "how did you hear about us" | Anecdotes count early — log them |
| Ranking movement on target pages | GSC position for commercial pages, studs/engagement collections | Post-activation baseline → movement by day 90 |
| Commercial/vendor relationships created (SGD) | CRM + CSV notes | 2–3 real PM/vendor-list placements |
| Content assets shipped | content-assets file | Per roadmap |
| Toxic/spam handled | Quarterly audit | Documented review, 0 panic-disavows |

## Measurement notes

- GSC/GA4 programmatic access is BUILT but awaiting owner activation (see
  handoff). Until then: manual GSC exports on request; do not fabricate numbers.
- A `won` link is counted only after verifying: live URL, expected rel attribute,
  sane anchor. Record all three in the CSV Evidence column.
- Every won link gets a source attribution note when a lead mentions it —
  over a year this builds the real ROI picture no tool provides.
